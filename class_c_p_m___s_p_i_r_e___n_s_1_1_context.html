Detailed Description

Generalized OpenGL context. With the existence of wrappers like http://code.google.com/p/angleproject/, writing OpenGL ES 2.0 compliant code means we can run using DirectX on windows (better driver support). All functions contained herein must be thread safe as they will be executed from the Spire's thread.
